Replace all zippyshare links to embed media.

== Description ==

This plugin converts all zippyshare links in your Wordpress content into zippyshare player. It's very simple. You can pick your colors in plugin settings page too :)

== Installation ==

1. Upload `simple-zippyshare-embed` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to plugin settings page and choose your colors and width :)
